# Base-Converter-Expression
Takes an input (Any base between 2 and 16) and converters that input to the desired base; read the README file to show you how it works.

HP PRIME BASE CONVERTER
=======================

A powerful Number Base Converter for the HP Prime Calculator. 
Unlike the built-in tools, this program supports FRACTIONAL numbers 
(e.g., 10.5, A.C) and provides step-by-step working.

INSTALLATION
------------
1. Open the HP Connectivity Kit on your PC.
2. Connect your calculator.
3. Right-click "Programs" and select "New Program".
4. Name the program "Baseconv".
5. Delete any existing code, paste the source code in, and save.

FEATURES
--------
- Supports any base from 2 to 16.
- Converts INTERGERS and FRACTIONS (e.g., converting 12.5 from Base 10 to Base 2).
- Shows step-by-step workings (Logic).
- Special high-speed mode for Hexadecimal to Binary.
- Adjustable precision for fractional results.

HOW TO USE
----------
1. Press [Program] and select "Baseconv".
2. Enter the Number you want to convert (e.g., "1F.8").
3. Enter the SOURCE Base (e.g., 16 for Hex).
4. Enter the TARGET Base (e.g., 10 for Decimal).
5. Enter Precision (how many decimal places to calculate).

EXAMPLE
-------
Convert "10.1" from Base 2 (Binary) to Base 10 (Decimal).

1. Run Program.
2. Number: 10.1
3. Input Base: 2
4. Output Base: 10
5. Result: 2.5

CONTROLS
--------
ENTER -> Continue / Confirm inputs
ESC   -> Exit the program after viewing results
